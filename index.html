Braille Board Braille Board

Accessible Communication for Everyone

Braille Board is a compact, portable keyboard module that magnetically attaches to your smartphone, enabling fast and intuitive text input through the Braille alphabet for people with visual impairments.

Get started

Braille Input

6 dedicated buttons for fast and intuitive Braille character input, enabling rapid text composition using the standard Braille alphabet.

02
Navigation

3 dedicated navigation buttons for moving through menus, confirming selections, and creating custom shortcuts for frequent actions.

03
MagSafe Attachment

Compact, lightweight design that magnetically attaches to your smartphone without requiring bulky external equipment.

Revolutionary Accessibility Technology

First-of-its-kind Braille input system designed specifically for smartphones. Braille Board enables fast, accurate text input and navigation without compromising independence or requiring bulky assistive technology.

Braille Board works seamlessly with any smartphone that supports MagSafe technology. Simply attach, pair via Bluetooth, and start typing. Compatible with iOS and Android devices, it integrates perfectly with all your favorite accessibility apps and features. Developed together with our Co-creators, this project is built hand in hand with the community. Portable design

The Braille Board is designed so you can always carry it with your phone without feeling any extra burden. Whether it's in your pocket or a small bag, you won't even notice you're carrying one more device.

Battery life

To make the device smaller and more convenient, yes, we had to sacrifice a little in battery size and capacity. But thanks to a special standby mode, we managed to make it last a full day. In short, even if you forget to turn it off, it won't waste energy.

Automatic software update

Through its dedicated app, the keyboard can automatically update its software, making it more user-friendly and universal.

Special application

The Braille Board's special app allows you to add the language you want to the keyboard. More than 100 languages are available. With a dedicated hot key, you can easily switch between multiple languages while using it.

Special hot keys

You can add your own custom hot keys to quickly open apps, call your loved ones, switch between applications and languages. Hot keys also help you finally avoid those uncomfortable voice message buttons.
